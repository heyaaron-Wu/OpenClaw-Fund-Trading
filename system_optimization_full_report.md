## 🟡 需要优化的功能

### 1️⃣ 13:35 候选池刷新 → 调整为按需刷新

**当前问题:**
- 每日刷新但很少使用
- 满仓状态下无实际意义
- 浪费 token 和 API 调用

**优化方案:**
```
方案 A: 改为每周三刷新一次
方案 B: 仅在持仓<3 只时自动刷新
方案 C: 完全删除，需要时手动触发

推荐：方案 B
```

---

### 2️⃣ 14:48 执行门控 → 简化为确认环节

**当前问题:**
- 与 14:00 决策功能重叠
- 仅异常时推送，使用频率低
- 增加系统复杂度

**优化方案:**
```
方案 A: 合并到 14:00 任务
方案 B: 保留但改为静默确认
方案 C: 删除

推荐：方案 B (保留最终确认环节)
```

---

### 3️⃣ 系统清理任务 → 调整为每周一次

**当前问题:**
- 每日凌晨 1:00 执行
- 实际无需每日清理
- 曾出现超时错误

**优化方案:**
```
方案 A: 改为每周日凌晨 3:00
方案 B: 改为手动触发
方案 C: 删除，使用系统自带清理

推荐：方案 A
```
