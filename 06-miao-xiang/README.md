# 妙想 API 使用记录

东方财富妙想 API 的调用记录、查询结果和分析摘要。

## 目录结构

```
06-miao-xiang/
├── README.md          # 本文件
├── daily/             # 每日妙想查询记录
├── weekly/            # 每周妙想使用回顾
├── queries/           # 原始查询日志（JSON）
└── analysis/          # 妙想数据驱动的分析报告
```

## 用途

- **晨间简报**：隔夜消息、市场预判
- **交易决策**：实时行情、板块分析
- **日终复盘**：持仓基金表现、市场回顾
- **周报复盘**：本周妙想查询汇总 + 数据质量评估

## 数据源优先级

AKShare (主) → Iwencai (备) → 妙想 API (备) → 本地缓存 (兜底)
