## 🎯 核心发现

### 1. Google Finance / Yahoo Finance
*   **现状**: Google 已关闭公开金融 API。Yahoo Finance 的 `yfinance` 库**不支持中国公募基金**（仅支持美股/ETF）。
*   **结论**: ❌ **不可用**于中国基金数据。

### 2. BaoStock (证券宝)
*   **现状**: 免费开源，但**主要覆盖股票**，公募基金数据覆盖极少。
*   **结论**: ⚠️ **不适用**于基金候选池。

### 3. Tushare Pro (强烈推荐)
*   **现状**: 国内最专业的开源金融数据社区。覆盖所有公募基金、股票、期货。
*   **优势**:
    *   ✅ 数据准确、更新快
    *   ✅ 提供基金净值、分红、持仓等详细数据
    *   ✅ API 响应快，支持批量查询
    *   ✅ 免费基础额度（需注册获取 Token）
*   **劣势**: 需要注册账号获取 Token，高级接口需积分。
*   **结论**: 🥈 **最佳补充数据源**

### 4. AKShare (当前核心)
*   **现状**: 已验证稳定可用，覆盖所有公募基金。
*   **优势**: 免费、无需注册、数据全面。
*   **结论**: 🏆 **保持为核心数据底座**

## 🚀 建议方案：三源冗余架构

为了达到**金融级高可用**，建议采用以下架构：

```
[ 请求基金数据 ]
       │
       ▼
┌─────────────┐
│ 1. AKShare  │ ← 主数据源 (免费、稳定)
└──────┬──────┘
       │ 失败/超时
       ▼
┌─────────────┐
│ 2. Tushare  │ ← 备用数据源 (需Token，高质量)
└──────┬──────┘
       │ 失败/无Token
       ▼
┌─────────────┐
│ 3. Iwencai  │ ← 辅助数据源 (自然语言查询)
└──────┬──────┘
       │ 失败
       ▼
   [ 使用缓存/跳过 ]
```
