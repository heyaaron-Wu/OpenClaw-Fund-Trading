## 💡 重要经验

1. **Iwencai API 不稳定**: 收益率数据常返回 0%，不可作为唯一数据源
2. **AKShare 是最佳底座**: 免费、稳定、无需 Token、覆盖所有公募基金
3. **多源降级架构必要**: 单一 API 失败不应导致系统中断
4. **智能挖掘有效**: AKShare 排行挖掘发现 4 只 75 分满分基金
5. **评分模型需要缓冲带**: 0% 断崖会导致微跌基金被误杀
6. **技能名称必须准确**: SkillHub 上名称与预期不同（如"问财选 ETF"不存在）
7. **美股与 A 股完全独立**: 不同的数据目录、Webhook、运行时间
8. **时区配置关键**: 美股任务使用北京时间，但逻辑对应美东时间
