## 📱 接收方式

- **渠道**: 企业微信 (wecom)
- **账户**: default
- **模式**: announce (自动发送到聊天频道)

---

## ⏰ 每个时间段会做什么

### 🌅 09:00 健康检查
系统会发送：
- 当前总资产市值
- 距离目标的进度（+X%）
- 今日市场概况
- 是否需要调整策略

### 🕑 14:48 最终决策
系统会发送：
- 是否建议买入新基金（BUY）
- 或维持现有持仓（HOLD）
- 如果建议 BUY，会给出基金代码和金额
- 你需要在 15:00 前手动下单并回复确认

### 🌃 20:05 日终复盘
系统会发送：
- 今日盈亏计算
- 持仓市值更新
- 当日操作总结
- 明日关注要点

### 📊 每周五 20:00 周报复盘
系统会发送：
- 本周总收益率
- 每周收益曲线图
- 目标进度百分比
- 下周策略调整建议

## 🔧 常用管理命令

### 查看定时任务状态
```bash
openclaw cron list --all
```

### 暂停某个任务（例如周末不发送）
```bash
openclaw cron disable fund-daily-check
```

### 恢复某个任务
```bash
openclaw cron enable fund-daily-check
```

### 立即测试发送（调试用）
```bash
openclaw cron run fund-daily-check
```

### 删除某个任务
```bash
openclaw cron rm fund-daily-check
```

### 添加新的定时任务
```bash
# 示例：每天凌晨 1 点推送简报
openclaw cron add \
  --name "fund-nightly" \
  --account "default" \
  --channel "wecom" \
  --agent "main" \
  --cron "0 1 * * *" \
  --description "基金挑战 - 每日凌晨简报" \
  --message "基金挑战夜间盘点"
```

## 💡 使用技巧

### 1. 临时关闭周末推送
如果你不想在周末收到消息，可以：
```bash
# 周六早上停用
openclaw cron disable fund-daily-check
openclaw cron disable fund-1448-decision
openclaw cron disable fund-2005-review

# 周日晚上重新启用
openclaw cron enable fund-daily-check
```

### 2. 调整汇报内容
如果你想修改某次推送的内容：
```bash
openclaw cron edit fund-daily-check \
  --description "基金挑战 - 自定义健康检查" \
  --message "自定义的健康检查内容"
```

### 3. 查看所有历史推送记录
```bash
openclaw cron runs fund-daily-check
```

## 🚨 注意事项

1. **工作时间**: 只在交易日 (周一至周五) 的 09:00-16:00 之间发送
2. **15:00 截止**: 下午的 BUY 决策需要在 15:00 前手动下单
3. **网络问题**: 如果推送失败，会在下次运行时重试
4. **隐私保护**: 所有推送只包含投资信息，不涉及个人隐私
