# 通用技能说明

本文档详细说明系统使用的通用技能（29 个）。

---

## 📊 财经数据（5 个）

| 技能名称 | 作用 | 使用场景 |
|----------|------|---------|
| `akshare-finance` | 财经数据接口库（股票/期货/基金/外汇/债券） | 获取金融市场数据 |
| `akshare-stock` | A 股量化数据分析（行情/财务/板块） | A 股股票查询/财务分析 |
| `etf-assistant` | ETF 投资助理 | 查询行情/筛选 ETF/对比分析/定投计算 |
| `finance-lite` | 每日宏观 + 市场简报 | FRED 数据 + 基准指数 + 自选股 |
| `stock-watcher` | 个人股票看板管理 | 添加/删除/监控自选股 |
