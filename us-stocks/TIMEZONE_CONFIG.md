# 🕐 美股系统时区配置说明

## ⏰ 时区对照

**系统时区**: 亚洲/上海 (UTC+8)  
**美股市场**: 美国东部时间 (UTC-5 / UTC-4)

### 时间转换表

| 事件 | 美东时间 | 北京时间 (UTC+8) |
|------|---------|-----------------|
| **夏令时** (3 月 -11 月) | | |
| 美股开盘 | 09:30 | 21:30 |
| 美股收盘 | 16:00 | 04:00 (次日) |
| **冬令时** (11 月 - 次年 3 月) | | |
| 美股开盘 | 09:30 | 22:30 |
| 美股收盘 | 16:00 | 05:00 (次日) |

---

## 🔄 Cron 任务时间 (北京时间)

### 已配置任务

| 任务 | Cron 表达式 | 北京时间 | 对应美东时间 | 说明 |
|------|-----------|---------|------------|------|
| **us-stock-selector** | `30 5 * * 1-5` | 05:30 | 17:30 (前一日) | 美股筛选 (收盘后) |
| **us-portfolio-manager** | `0 6 * * 1-5` | 06:00 | 18:00 (前一日) | 持仓管理 |
| **us-daily-review** | `0 7 * * 1-5` | 07:00 | 19:00 (前一日) | 日终复盘 |

### 设计逻辑

```
美股交易日 (美东时间):
  09:30 开盘
  16:00 收盘
  ↓
北京时间 (次日):
  04:00/05:00 美股收盘
  05:30 运行筛选 (获取完整当日数据)
  06:00 持仓分析
  07:00 日终复盘 + 飞书推送
```

## 📅 运行示例

### 周一 (美东时间) → 周二 (北京时间)

```
美东时间 周一 16:00 美股收盘
          ↓
北京时间 周二 04:00/05:00 (夏令时/冬令时)
          ↓
北京时间 周二 05:30 运行美股筛选
北京时间 周二 06:00 运行持仓管理
北京时间 周二 07:00 运行日终复盘 + 推送
```

---

## ⚠️ 注意事项

### 夏令时 vs 冬令时
- **夏令时** (3 月第二个周日 - 11 月第一个周日): 北京时间 = 美东时间 + 12 小时
- **冬令时** (11 月第一个周日 - 3 月第二个周日): 北京时间 = 美东时间 + 13 小时

### 周末和节假日
- 美股周末休市
- Cron 配置为 `* * 1-5` (周一至周五)
- 美股节假日自动跳过 (无交易数据)

### 数据更新
- 筛选数据来自 Iwencai API (T+0 或 T+1)
- 05:30 运行可获取完整的昨日收盘数据
- 避免在盘中运行 (数据不完整)
