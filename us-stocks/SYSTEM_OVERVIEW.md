## 📊 美股筛选功能

### 4 大筛选维度
| 类型 | 筛选标准 | 预期数量 |
|------|---------|---------|
| **价值股** | PE<20, PB<3, 股息>2% | 20-30 只 |
| **成长股** | 营收增长>20%, 利润增长>25% | 20-30 只 |
| **科技股** | AI/芯片/半导体/云计算 | 20-30 只 |
| **动量股** | 近 1 月涨幅>15%, 成交量放大 | 15-25 只 |

### 综合评分算法
```python
综合评分 = 基础分 (50) + 因子加分

# 价值股因子
- PE < 15: +15 分
- 股息率 > 3%: +10 分

# 成长股因子
- 营收增长 > 30%: +20 分
- 利润增长 > 40%: +20 分

# 科技股因子
- 科技行业：+10 分

# 动量股因子
- 近 1 月涨幅 > 20%: +15 分

# 上限：100 分
```

## 💼 持仓管理功能

### 功能列表
- ✅ 持仓加载/保存
- ✅ 盈亏计算
- ✅ 交易信号生成 (BUY/SELL)
- ✅ 持仓分析
- ✅ 价格更新

### 交易信号逻辑
```python
# 买入信号
- 综合评分 > 70
- 不在当前持仓中
- 来自股票池 Top 20

# 卖出信号
- 亏损 > 10%
- 基本面恶化
- 有更好的替代标的
```

## 📝 日终复盘功能

### 复盘内容

## 🎯 使用流程

### 日常流程
```
10:00 📊 美股筛选
      ↓
      生成股票池 (80-100 只)
      ↓
14:00 人工选股 (从股票池中选择)
      ↓
      执行交易 (手动/自动)
      ↓
22:00 💼 持仓管理
      ↓
      分析持仓 + 生成信号
      ↓
23:30 📝 日终复盘
      ↓
      生成报告 + 飞书推送
```

### 手动操作
```bash
# 运行筛选
python3.11 us_stock_selector_enhanced.py

# 运行持仓管理
python3.11 portfolio_manager.py

# 运行日终复盘
python3.11 daily_review_enhanced.py
```
