## 📝 版本号规则

### 版本号格式

```
v主版本。次版本。修订号
例如：v1.1.1
```

### 自动升级规则

**系统根据更新内容自动决定版本号：**

| 更新内容 | 版本号变化 | 示例 |
|---------|-----------|------|
| **新功能、重大功能增强** | 次版本 +1 | v1.0.x → v1.1.0 |
| **新功能延续、小功能添加** | 修订号 +1 | v1.1.0 → v1.1.1 |
| **Bug 修复、文档更新** | 修订号 +1 | v1.0.18 → v1.0.19 |

**判定逻辑：**
```
1. 检查是否有新功能 → 次版本 +1（v1.1.0）
2. 检查是否新功能延续 → 修订号 +1（v1.1.1）
3. 否则 → 修订号 +1（v1.0.19）
```
