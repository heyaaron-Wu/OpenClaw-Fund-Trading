## 🚀 核心功能新增

### 1. 东方财富妙想 API 集成

#### 资讯获取能力
- ✅ **盘后收评** - 15:30 自动获取当日收评资讯
- ✅ **多源 fallback** - 收评→复盘→要闻，智能切换
- ✅ **AI 总结** - 提取核心要点，不是复制粘贴
- ✅ **来源链接** - 每条资讯附带可点击链接
- ✅ **时间显示** - 显示资讯发布时间

#### API 查询策略
```bash
1. 查询"${TODAY} A 股收评" → 失败？
2. 查询"${TODAY} 复盘" → 失败？
3. 查询"${TODAY} 收盘点评" → 失败？
4. 查询"${TODAY} A 股市场" → 失败？
5. 查询"${TODAY} 重要新闻" → ✅ 成功
```

## 📊 日终复盘系统升级

### 报告模板优化

**之前格式**:
```markdown
# 基金挑战日终复盘
## 📊 今日盈亏
...
```

**现在格式**:
```markdown
# 日终复盘 YYYY-MM-DD

## 📊 今日盈亏
## 📈 持仓明细
## 📰 今日要闻（东方财富）← 新增
## 📝 今日总结
## 📅 挑战进度
```

### 新增章节

#### 📰 今日要闻（东方财富）
- 5 条盘后收评
- AI 总结核心要点
- 来源 + 时间 + 可点击链接

**示例**:
```markdown
**1. 沪指涨 0.24%**
超 2800 只个股上涨，有色、农业拉升
*来源：[证券日报网](链接)* | 2026-03-30 16:05:00
```

#### 📝 今日总结
- 市场表现（上证指数/创业板指/科创 50）
- 亏损原因分析（市场/板块/持仓）
- 后市展望（短期/中期/操作建议）

#### 📅 挑战进度
- 挑战开始日期
- 当前天数
- 目标金额（1300 元）
- 距离目标
